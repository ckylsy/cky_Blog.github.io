$cmd=’\‘;

输出yes，就是没有过滤\

问题出现在前面的两个\，因为正则表达式是通过了两层解析，一层是php的，一层是正则的

前面两个\，经过php解析器解析后成为了\，|因为是php中的保留符号，所以需要一个转义\来匹配，所以猜测应该就对|\的匹配，尝试，输入$cmd=|\,yes
